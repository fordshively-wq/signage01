# signage01

SignalBoard is a Node-based digital signage platform prototype with accounts, signage groups, display pairing codes, calendars, weather, media, countdowns, blackout schedules, and a lightweight room controller.

## Run locally

```bash
npm start
```

Then open:

- `http://localhost:3000`
- `http://localhost:3000/dashboard`
- `http://localhost:3000/display`
- `http://localhost:3000/control`

## Notes

Local account and signage data is stored in `data/db.json`, which is ignored by git.
